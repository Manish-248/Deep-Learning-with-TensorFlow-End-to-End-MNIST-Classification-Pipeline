# 🤖 TensorFlow Deep Learning: MNIST Classification

## 📌 Overview

This project demonstrates a complete deep learning workflow using **TensorFlow/Keras** to classify handwritten digits from the MNIST dataset.

---

## 🚀 Features

* Data preprocessing and normalization
* Deep Neural Network with dropout regularization
* Model training with validation
* Performance visualization
* Model saving & loading
* Prediction demo

---

## 🧠 Model Architecture

* Dense (256, ReLU)
* Dropout (0.3)
* Dense (128, ReLU)
* Dropout (0.3)
* Dense (10, Softmax)

---

## 📊 Dataset

* MNIST Handwritten Digits
* 60,000 training samples
* 10,000 test samples

---

## 📈 Results

* High accuracy on test dataset
* Stable training and validation curves

---

## 🛠️ Tech Stack

* TensorFlow / Keras
* NumPy
* Matplotlib
